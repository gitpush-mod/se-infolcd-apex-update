This is an addon for [url=https://steamcommunity.com/sharedfiles/filedetails?id=3572467804]Not Just For Looks[/url]. It extends NJFL by filling in missing tiers and adding new block ladders for DLC content that vanilla didn't ship tiered variants for.

[hr][/hr]

[b]Flat Atmospheric Thrusters - Base + Upgraded tiers[/b]

NJFL gives the Flat Atmospheric Thrusters from Warfare Evolution an [u]Advanced[/u] tier. This addon fills in the missing [u]Base[/u] and [u]Upgraded[/u] tiers, so the variant arrows on flat atmo thrusters cycle through all three.

[quote]
With both mods installed, the variant arrows on Flat Atmospheric Thrusters cycle:
[list]
[*][b]Base[/b] - vanilla cost, vanilla force, no Superconductor tax. Effectively just a renamed vanilla flat atmo thruster.
[*][b]Upgraded[/b] - 2x component cost, 4x force (~2x thrust to mass ratio), vanilla power. Superconductor tax scaled by block size: 10 / 5 / 4 / 2 SC for L-grid Large / L-grid Small / S-grid Large / S-grid Small.
[*][b]Advanced[/b] - 3x component cost, 9x force (~3x thrust to mass ratio), vanilla power. SC + Reactor Component tax. (Provided by Not Just For Looks.)
[/list]
Covers all 8 flat atmo variants: Large/Small thruster size, L-grid/S-grid, Flat/D-Shape geometry.
[/quote]

[b]Without Not Just For Looks installed[/b], you still get the Base and Upgraded tiers as new blocks - just no Advanced tier on the variant arrow cycle.

[hr][/hr]

[b]Bulk Cargo Containers - Base + Upgraded + Advanced tiers[/b]

The Economy 2 DLC ships three Bulk Cargo Container visual variants (A / B / C) that are functionally identical to each other. This addon re-tiers them into a [u]Base[/u] / [u]Upgraded[/u] / [u]Advanced[/u] storage ladder, with inventory volumes benchmarked against the NJFL Upgraded Industrial Cargo Container (~844 m^3) as the reference.

[quote]
The variant arrows on Bulk Cargo Containers now cycle three real tiers:
[list]
[*][b]Bulk Cargo Container[/b] (Base, was visual variant "B") - 1.5x vanilla bulk components, 15 SC tax, 1.5x reference inventory (~1266 m^3)
[*][b]Upgraded Bulk Cargo Container[/b] (was visual variant "A") - 3x vanilla bulk components, 20 SC + 5 Reactor Component tax, 3x reference inventory (~2531 m^3, 2x the Base tier)
[*][b]Advanced Bulk Cargo Container[/b] (was visual variant "C") - 4.5x vanilla bulk components, 25 SC + 10 Reactor Component tax, 4.5x reference inventory (~3797 m^3, 3x the Base tier)
[/list]
All three are L-grid Large blocks (6x3x3 footprint). Critical component on all tiers is Superconductor.

The NJFL Upgraded Industrial Cargo Container is [u]untouched[/u] and remains the denser-per-cell option for cramped builds - same Superconductor gate, smaller 3x3x3 footprint. Pick the long-and-roomy Bulk variants for stations with space, or the compact Industrial cube for tighter ship/station builds.
[/quote]

The Bulk Cargo Container ladder works whether or not Not Just For Looks is installed - all three tiers live in this mod.

[quote]
[b]Quick heads up on compatibility:[/b]
This mod adds new thruster and cargo container blocks and overrides the vanilla Atmospheric Thruster variant group to make the flat atmo tiers cycle properly. If another mod also restructures atmospheric thruster groups or the bulk cargo container subtypes, your mod load order will matter.
[/quote]
