# se-infolcd-apex-update

**InfoLCD for Apex LCD Blocks** — Space Engineers 1 mod. Text surface scripts
that render real-time ship + base data on Apex LCD screens: cargo, power,
jump-drive charge, ammo, airlock status, and more.

[Chris](https://github.com/Godimas101)'s flagship SE mod. Client-side only.
1,335+ Steam Workshop subscribers.

## Sibling variant

A more feature-rich sibling lives at
[`gitpush-mod/se-infolcd-apex-advanced`](https://github.com/gitpush-mod/se-infolcd-apex-advanced).
The two variants should be kept in sync on shared features.
